# 네이버 스마트스토어 제휴 제안서

## 생애주기 맞춤형 라이프 로깅 플랫폼 '위빗(Webit)' × 네이버 스마트스토어 커머스 연동 제안

---

## 1. 제안 배경

네이버 스마트스토어는 국내 최대 이커머스 인프라로서, 웨딩·리빙 카테고리에서 이미 강력한 셀러 생태계를 보유하고 있습니다. 그러나 **결혼 준비 소비자의 구매 여정은 정보 탐색(블로그/카페) → 비교 → 구매로 분절**되어 있으며, 각 단계의 데이터가 연결되지 않아 정밀한 타겟팅과 전환 최적화에 한계가 존재합니다.

위빗(Webit)은 커플의 연애-결혼-가족 생애주기 데이터를 누적하는 플랫폼으로, **"언제, 무엇을, 왜 사는지"에 대한 맥락 데이터**를 보유합니다. 이를 네이버 스마트스토어의 커머스 역량과 결합하면, 웨딩 카테고리에서 기존 대비 높은 구매 전환을 기대할 수 있습니다.

---

## 2. 위빗(Webit) 서비스 개요

### 핵심 컨셉
연애부터 결혼, 가족까지 — 생애주기를 관통하는 커플 라이프 로깅 플랫폼

### 서비스 구조

| 단계 | 주요 기능 | 수집 데이터 |
|------|----------|------------|
| **Phase 1: 연애** | 커플 캘린더, 포토 갤러리, D-day, 여행 플래너 | 데이트 장소, 선물 이력, 여행 선호, 기념일 패턴 |
| **Phase 2: 결혼 준비** | 웨딩 예산 관리, 업체 관리, 디지털 청첩장, 하객 관리 | 예산 규모, 선호 스타일, 구매 카테고리, 일정 |
| **Phase 3: 가족** | 가족 아카이브, 공유 캘린더, 포토 정리 | 라이프스타일 변화, 육아 관련 소비 패턴 |

### 차별점
- **데이터 연속성**: 연애 시절부터 축적된 취향 데이터가 결혼 준비 단계에서 구매 추천의 근거로 활용
- **자연스러운 커머스 전환**: 예산 관리, 업체 비교 기능 내에서 상품 추천이 "유용한 정보"로 수용되는 UX 설계
- **광고 없는 구간(Phase 1, 3) → 상업 구간(Phase 2) 분리**: 유저 신뢰를 먼저 확보한 뒤 커머스를 노출하는 전략

## 3. 제휴 제안 내용

### 3-1. 네이버 스마트스토어 상품 연동

**연동 방식**
- 위빗 앱 내 결혼 준비 단계(Phase 2)에서 네이버 스마트스토어 웨딩 카테고리 상품을 큐레이션하여 노출
- 네이버 커머스 API 연동을 통해 상품 정보, 가격, 리뷰를 실시간 반영
- 유저가 위빗 내에서 상품을 탐색하고, 네이버 스마트스토어로 이동하여 결제 완료

**큐레이션 대상 카테고리**
- 스드메(스튜디오/드레스/메이크업)
- 청첩장 및 디지털 청첩장
- 웨딩 소품 및 굿즈 (포스터, 거울, 액자 등)
- 예단/예물
- 혼수 가전 및 리빙
- 신혼여행 관련 상품

### 3-2. 데이터 기반 타겟팅 가치

위빗이 네이버 스마트스토어에 제공할 수 있는 핵심 가치는 **구매 맥락 데이터**입니다.

| 기존 타겟팅 | 위빗 데이터 기반 타겟팅 |
|------------|---------------------|
| 키워드 검색 기반 (의도 추정) | 실제 결혼 일정·예산 확정 유저 |
| 인구통계 기반 (20-30대 여성) | 결혼 D-180, 예산 3,000만원, 모던 스타일 선호 유저 |
| 과거 구매 이력 기반 | 연애 시절 선호 브랜드 + 결혼 준비 단계 진입 시점 데이터 |

### 3-3. 제안 제휴 모델

| 모델 | 내용 | 네이버 측 이점 |
|------|-----|--------------|
| **CPS (Cost Per Sale)** | 위빗 경유 구매 건당 수수료 정산 | 성과 기반 비용, 리스크 최소화 |
| **큐레이션 광고** | 위빗 내 스마트스토어 셀러 상품 프리미엄 노출 | 신규 트래픽 채널 확보 |
| **데이터 제휴** | 비식별 웨딩 트렌드 리포트 공유 | 웨딩 카테고리 MD 전략 수립 지원 |

## 4. 시장 기회

### 국내 웨딩 시장 규모
- 국내 연간 혼인 건수: 약 19만 건 (2024년 통계청 기준)
- 평균 결혼 비용: 약 2.4억 원 (양가 합산, 한국소비자원)
- 웨딩 관련 온라인 소비 비중 지속 증가 (특히 스드메, 청첩장, 혼수 카테고리)

### 타겟 유저 특성
- **고관여**: 결혼 준비 기간 평균 6-12개월간 집중적 소비
- **고객단가**: 단일 이벤트 기준 국내 최고 수준의 객단가
- **정보 탐색 적극적**: 블로그, 카페, 앱을 통한 비교·검색 행동이 활발

기획자 본인이 직접 결혼 준비를 경험하며, 기존 웨딩 앱의 불편함과 데이터 단절 문제를 1차적으로 검증했습니다. 사용자 시나리오가 추상적 기획이 아닌, 실제 경험에서 도출된 Pain Point에 기반합니다.
